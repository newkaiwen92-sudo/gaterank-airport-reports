## 最新月度报告

最新归档报告见下方报告目录。将 Markdown 报告放入 `incoming/` 后运行：

```bash
python3 scripts/import_report.py --incoming incoming
python3 scripts/update_readme.py
```

脚本会自动生成：

- `reports/YYYY-MM/<slug>.md`
- `data/reports-index.json`
- README 月度报告索引
